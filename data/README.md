# Data Folder

This folder contains the data files used by the Tableau project.

## Raw Data

- `raw/house_price_50k.csv`: original project dataset.

## Processed Data

- `processed/house_price_tableau.csv`: prepared market file used for the Tableau market overview. It keeps the full 50,000-row market dataset.

## Derived Data

- `derived/model_predictions_tableau.csv`: 10,000-row model holdout prediction file.
- `derived/model_metrics_tableau.csv`: model performance metrics.
- `derived/feature_importance_tableau.csv`: feature importance evidence.
- `derived/location_income_summary.csv`: market summary by location and income level.
- `derived/data_dictionary.csv`: field descriptions for the Tableau files.

The dataset is synthetic and uses unspecified `price units`.
