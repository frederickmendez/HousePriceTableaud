# Tableau Workbook Notes

Workbook files:

- `tableau/workbook/house_price_tableau_dashboard.twb`
- `tableau/workbook/house_price_tableau_dashboard.twbx`

The `.twbx` file is the packaged Tableau workbook. It is the preferred file to open because it keeps the workbook and packaged supporting data together.

## Dashboard Areas

- Market overview: uses the 50,000-row prepared market file.
- Prediction reliability: uses the 10,000-row model holdout prediction file.
- Tableau story: presents seven narrative points from market context to recommendations.

## Evidence Caveats

- The dataset is synthetic.
- Price values are shown as unspecified `price units`.
- The workbook should not be interpreted as a live real-estate valuation system.
- The screenshots are static evidence of final Tableau states.
